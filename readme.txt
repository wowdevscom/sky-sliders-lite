Beautiful, ready-to-use Elementor slider widgets — no coding required. Add stunning image sliders to any Elementor page in seconds.

== Description ==

**Sky Sliders** gives you professionally designed Elementor slider widgets that you can drop onto any page, customize with Elementor's visual editor, and publish — no code, no shortcodes, no hassle.

Each widget is built on [Swiper.js](https://swiperjs.com/), the most widely used touch slider library, so your sliders are smooth, responsive, and mobile-friendly out of the box.

= Free Widgets Included =

**Minimal Slider**
A clean, distraction-free full-screen slider built for portfolios, agencies, and creative showcases. Each slide supports a sub-title, headline, body text, call-to-action button, image, and optional video URL. Navigation arrows, dot pagination, autoplay, loop, and transition speed are all configurable from the Elementor panel.

**Photography Slider**
An immersive, image-first slider designed for photographers, studios, and visual storytellers. Each slide features a two-part title (normal + accent span), an outline text layer for dramatic depth, a background image, and a clickable link. A built-in social links sidebar and slide pagination let visitors stay oriented while they browse your work.

= Why Sky Sliders? =

* **Drag-and-drop** — find the widget in the Elementor panel and place it anywhere on your page.
* **Fully responsive** — every slider adapts perfectly to desktop, tablet, and mobile screens.
* **Touch & swipe ready** — powered by Swiper.js for native-feeling touch support on all devices.
* **Pixel-perfect styling** — control colors, typography, spacing, height, and more directly inside Elementor.
* **Autoplay & loop** — set independent autoplay delay and slide transition speed per widget.
* **Navigation & pagination** — toggle arrows and dot indicators on or off individually.
* **Title HTML tags** — choose H1–H6, div, span, or p for correct heading hierarchy and SEO.
* **No page builders lock-in** — widgets live inside Elementor; uninstalling restores your site cleanly.
* **Lightweight** — per-widget CSS and JS are loaded only on pages that actually use the widget.

== Installation ==

**Option 1 — WordPress Plugin Directory (recommended)**

1. Go to **Plugins → Add New** in your WordPress dashboard.
2. Search for **Sky Sliders**.
3. Click **Install Now**, then **Activate**.

**Option 2 — Manual upload**

1. Download the plugin ZIP from the WordPress Plugin Directory.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Choose the ZIP file and click **Install Now**, then **Activate**.

**Option 3 — FTP**

1. Extract the ZIP and upload the `sky-sliders-lite` folder to `/wp-content/plugins/`.
2. Activate the plugin from the **Plugins** screen in WordPress.

**Using the widgets**

1. Open any page or post in Elementor.
2. Search for **"Minimal"** or **"Photography"** in the Elementor widget panel.
3. Drag the widget onto your layout.
4. Add your slides, images, titles, and links in the Content tab.
5. Style colors, typography, and spacing in the Style tab.
6. Publish.

== Frequently Asked Questions ==

= Does this plugin work without Elementor? =

No. Sky Sliders is an Elementor add-on and requires Elementor (free or Pro) to be installed and active. The widgets will not appear without it.

= Which version of Elementor is required? =

Elementor 3.0.0 or later. Both the free and Pro versions of Elementor are supported.

= Will my sliders work on mobile? =

Yes. All Sky Sliders widgets are built on Swiper.js and are fully responsive and touch-swipe enabled on mobile and tablet devices.

= Can I use Sky Sliders and Sky Sliders Pro at the same time? =

No — installing Sky Sliders Pro automatically deactivates the Lite plugin to avoid conflicts. All your existing content is preserved and Pro is fully backward-compatible.

= How do I add autoplay to a slider? =

Select the widget in Elementor, go to the **Settings** tab in the widget panel, and toggle **Autoplay** on. You can then set the **Autoplay Speed** (delay between slides) and **Slide Speed** (transition duration) independently.

= Can I change the slide height? =

Yes. Each widget has a **Slider Height** control in its Additional settings section. You can set any value in px, vh, em, or rem.

= Does Sky Sliders add anything to pages that don't use the widget? =

No. Styles and scripts are registered globally but only enqueued by Elementor on pages that contain at least one Sky Sliders widget, keeping your site fast everywhere else.

= Where can I report a bug or request a feature? =

Please open an issue on our [GitHub repository](https://github.com/wowdevscom/sky-sliders) or use the WordPress.org support forum for this plugin.
