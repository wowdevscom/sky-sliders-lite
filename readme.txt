Free Elementor slider widgets from Sky Sliders.

== Description ==

Sky Sliders Lite provides the free widget set generated from the Sky Sliders source repository.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sky-sliders-lite` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the Plugins screen in WordPress.
3. Use the Sky Sliders widgets in Elementor.
